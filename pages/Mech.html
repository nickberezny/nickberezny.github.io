Virtual Gait Rehabilitation Robot Lite (ViGRR-LITE)
A 1-DOF rehabilitation robot designed for assisting the lower-limb for bed-bound stroke patients. Includes haptic capabilities for interaction with a virtual environment. A real-time control system using RT-Linux with a pre-empted kernel is used to implement force and impedance control for safe human interaction. Designed in collaboration with physiotherapists/doctors to ensure compatibility with the hospital setting.
