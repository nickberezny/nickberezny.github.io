Virtual Gait Rehabilitation Robot Lite (ViGRR-LITE)

A 1-DOF rehabilitation robot designed for assisting the lower-limb for bed-bound stroke patients. Includes haptic capabilities for interaction with a virtual environment. A real-time control system using Linux with a pre-empted kernel is used to implement force and impedance control for safe human interaction. Designed in collaboration with physiotherapists/doctors to ensure compatibility with the hospital setting.

Estimating Limb Impedance

Estimating limb impedance can be important for safe and effective physiotherapy. It can be used to predict spasticity or design adaptive controllers for human interaction. Here I use OpenSim to train a neural network that predicts leg end-point stiffness based on limb kinematics. Domain Adaption is then applied to tune the network to an individual based on a small amount of experimental data (1-3 points). The result is a reasonably accurate model trained on minimal amounts of data, which is desired in clinical settings.
